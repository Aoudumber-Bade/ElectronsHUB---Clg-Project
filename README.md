<h1 align="center">
  ElectronsHUB - College Major Project
</h1>


ElectronsHUB is a dynamic website developed by a team of four college students as a major project. This comprehensive platform serves as a hub for various electronic products, including smartphones 📱, smart TVs 📺, laptops and PCs 💻, wearable devices ⌚, audio devices 🎧, digital cameras 📷, and smart home devices 🏠. Built using HTML, CSS, JavaScript, PHP, and MySQL, ElectronsHUB offers a responsive design and user-friendly functionalities.


## ✨ Features

- ⚡️ Admin panel for managing categories and posts.
- ✅ Dynamic category sections with subcategories for electronic products.
- 🔒 User authentication system with login and registration functionalities.
- 📝 Guest posting feature allowing registered users to contribute content
- 📖 Blog section powered by CKEditor for easy post creation and management.
- 📝 Contact form designed to directly email submissions to the admin.
- 📱 Responsive design


## 👋 Getting Started - How to Run the Project with XAMPP

Follow these steps to run the project locally on your machine:

1. Clone the Repository

```bash
git clone https://github.com/Aoudumber-Bade/ElectronsHUB---College-Major-Project.git
```
2. Set Up XAMPP:
 - Download and install XAMPP from the official website.
 - Start the Apache and MySQL modules in XAMPP Control Panel.

3. Import the Database:
  - Locate the SQL file containing the database schema and data.
  - Open phpMyAdmin by visiting http://localhost/phpmyadmin in your web browser.
  - Create a new database for the project and import the SQL file into it.

4. Configure the Project:
  - Navigate to the project directory in your XAMPP installation (htdocs directory).
  - Update database connection settings in configuration files to match your XAMPP MySQL configuration.

5. Access the Website:
  - Open your web browser and navigate to http://localhost/<project_folder>.

6. Test the Website:
  - Thoroughly test the website's functionality to ensure everything is working as expected.
